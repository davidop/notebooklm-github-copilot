# Presales proposal generation

Use the `notebooklm` MCP server.

Target notebook: `<customer-or-opportunity-notebook>`

Question:

```text
Summarize the opportunity context, business objectives, technical requirements, constraints, risks, assumptions, and open questions. Cite each source-backed item.
```

Then produce:

1. Executive summary
2. Proposed Microsoft Cloud architecture
3. Delivery approach
4. Roles and team composition
5. Indicative timeline
6. Cost assumptions
7. Risks and mitigations
8. Open questions
