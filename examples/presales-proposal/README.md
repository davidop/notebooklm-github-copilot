# Example: Presales Proposal from NotebookLM

This example shows how to use GitHub Copilot + NotebookLM MCP to generate a presales proposal section grounded in past proposals and a service catalogue stored in NotebookLM.

## Scenario

A team is responding to an RFP for an Azure data migration. They have past proposals and a service catalogue in NotebookLM and want to reuse approved language.
