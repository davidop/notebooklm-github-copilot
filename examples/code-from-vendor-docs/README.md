# Example: Generate Code from Vendor Documentation

This example shows how to use GitHub Copilot + NotebookLM MCP to generate code that is grounded in vendor or API documentation stored in NotebookLM.

## Scenario

A team needs to integrate with a vendor's REST API. The vendor's API documentation and integration guide are stored in NotebookLM. The team wants to generate a typed Python client that matches the documented endpoints and data models.
