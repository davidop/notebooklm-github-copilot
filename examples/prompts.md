# Example prompts

## List notebooks

```text
Use the notebooklm MCP server to list my available notebooks. Return notebook name, likely purpose, and any ambiguity.
```

## Ask for grounded requirements

```text
Use the notebooklm MCP server. In the customer architecture notebook, answer with citations: what are the mandatory requirements for identity, network, monitoring, backup, and governance?
```

## Generate an implementation plan

```text
Use NotebookLM to retrieve the source-backed requirements for the Azure landing zone. Then create an implementation backlog with epics, user stories, acceptance criteria, and dependencies.
```

## Validate code against documents

```text
Use NotebookLM to get the expected behavior for the ingestion pipeline. Compare it with the current repository implementation and list gaps with file references.
```

## Proposal reuse

```text
Use NotebookLM to find source-backed proposal language for Azure Virtual Desktop, Microsoft Defender for Endpoint, and Azure governance. Produce a reusable proposal section in Spanish.
```
