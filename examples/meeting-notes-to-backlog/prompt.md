# Prompt Used

```text
Use NotebookLM. In the "discovery-workshop-notes" notebook, answer with citations:
1. What problems or pain points did stakeholders describe?
2. What features or capabilities were requested?
3. What action items were committed to?
4. What decisions were made during the workshop?

Then convert all findings into user stories in this format:
"As a [persona], I want [capability] so that [benefit]."

Group the stories by theme or epic. Add 2–3 acceptance criteria per story.
Flag any action items that are not user stories (technical spikes, decisions needed).

Return: user story list with acceptance criteria, epic groupings, source citations, and ambiguities that need clarification.
```
