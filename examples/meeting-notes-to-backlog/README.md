# Example: Convert Meeting Notes to Sprint Backlog

This example shows how to use GitHub Copilot + NotebookLM MCP to convert uploaded meeting notes into a structured sprint backlog.

## Scenario

A team has uploaded notes from a two-day discovery workshop to NotebookLM. They want to convert the action items, pain points, and feature requests into user stories for sprint planning.
