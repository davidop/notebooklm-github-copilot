# Sample Sources

> **DISCLAIMER:** These files are fictional demo sources and do not represent a real customer, organisation, vendor, or project. All names, figures, requirements, and technical details are invented for demonstration purposes only.

## Purpose

These files are designed to be loaded into a NotebookLM notebook as demo sources. They produce realistic-looking outputs when used with the demo prompts in [demo/sample-prompts.md](../demo/sample-prompts.md).

## How to use

1. Follow the instructions in [demo/sample-notebook-sources.md](../demo/sample-notebook-sources.md) to load these files into NotebookLM.
2. Use the prompts from [demo/sample-prompts.md](../demo/sample-prompts.md) in your Copilot client.
3. Compare your live outputs to the pre-generated samples in [demo/sample-outputs/](../demo/sample-outputs/).

## Do not use for real projects

These files contain invented data. Do not use them as a substitute for real project documentation.
