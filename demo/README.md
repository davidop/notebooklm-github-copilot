## Quick start

1. Load the sources described in [sample-notebook-sources.md](sample-notebook-sources.md) into a NotebookLM notebook.
2. Open VS Code with GitHub Copilot and the MCP server configured.
3. Choose a script from the table above.
4. Copy prompts from [sample-prompts.md](sample-prompts.md) and paste into Copilot chat.

## Audience guide

| Audience | Recommended script |
|----------|--------------------|
| Executive / stakeholder | 2-minute script |
| Developer or architect | 5-minute script |
| Workshop or training session | 15-minute script |
| Recorded video | 5 or 15-minute script + recording checklist |
