# Support

## How to get help

- **Documentation:** Start with the [docs/](docs/) folder. The [setup guide](docs/setup.md), [usage guide](docs/usage.md), [FAQ](docs/faq.md), and [troubleshooting guide](docs/troubleshooting.md) cover the most common questions.
- **GitHub Discussions:** For questions, ideas, and general discussion, open a [GitHub Discussion](https://github.com/davidop/notebooklm-github-copilot/discussions).
- **GitHub Issues:** For bug reports and feature requests, open a [GitHub Issue](https://github.com/davidop/notebooklm-github-copilot/issues). Use the provided issue templates.

---

## What is NOT supported

- **Production SLA** — This is a community project with no uptime or response time guarantees.
- **Enterprise support contracts** — No paid enterprise support is available for this repository.
- **notebooklm-mcp issues** — Issues with the underlying `notebooklm-mcp` package should be reported to its maintainers, not here.
- **Google NotebookLM issues** — Service availability, quota limits, and feature changes in NotebookLM are outside our control.
- **GitHub Copilot issues** — For Copilot-specific issues, contact GitHub Support.
