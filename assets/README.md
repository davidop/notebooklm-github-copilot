# Assets

This folder contains static assets for the project.

## social-preview.svg

A social preview image (1280×640) for use as the repository's Open Graph image on GitHub.

To set it as the repository social preview:
1. Go to the repository **Settings**.
2. Under **Social preview**, click **Edit**.
3. Upload `assets/social-preview.svg` (convert to PNG first if required — GitHub recommends PNG/JPG for social previews).

The SVG uses only text and geometric shapes. It contains no logos, trademarks, or copyrighted images.
