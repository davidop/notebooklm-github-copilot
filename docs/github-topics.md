## How to add topics

1. Go to the repository on GitHub.
2. Click the **gear icon** (⚙) next to the **About** section on the right side of the repository page.
3. In the **Topics** field, type each topic and press **Enter** or **Space** to confirm it.
4. Click **Save changes**.

GitHub recommends a maximum of **20 topics** per repository. The list above contains 19 topics, leaving room for one project-specific addition.
