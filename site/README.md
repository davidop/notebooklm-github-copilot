# Site

This folder contains the source files for the GitHub Pages site for this repository.

## How it works

GitHub Actions deploys this folder to GitHub Pages using the workflow in `.github/workflows/pages.yml`.

## Files

| File | Page |
|------|------|
| index.md | Homepage |
| getting-started.md | Getting started guide |
| clients.md | Client configuration guide |
| recipes.md | Recipes overview |
| security.md | Security overview |
| governance.md | Governance overview |
| evaluations.md | Evaluations overview |
| demo.md | Demo page |
| faq.md | Frequently asked questions |

## Local preview

To preview the site locally, install Jekyll and run:

```bash
cd site
bundle exec jekyll serve
```

Or simply browse the Markdown files directly in this folder.
