# Recipes

Ready-to-use workflow patterns for common engineering tasks using NotebookLM + GitHub Copilot.

## Core recipes

Core recipes live in the `recipes/` directory at the repository root.

| Recipe | Description |
|--------|-------------|
| Generate ADR | Draft architecture decision records from sources |
| Create backlog from meeting notes | Convert meeting notes to user stories |
| Generate presales proposal section | Draft proposal content from customer brief |
| Compare proposals | Compare two proposals against requirements |
| Source triangulation | Cross-reference multiple sources for consistent answers |

Browse all core recipes in the [recipes/ folder](../recipes/).
